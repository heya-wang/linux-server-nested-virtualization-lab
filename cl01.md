# CL01 Configuration
---
## CL01 Linux Base Setup
OS: Debian 13 (Bookworm)
Hostname: CL01
Virtualization: Hyper-V
VM Specs: 2 CPU / 2GB RAM / 20GB Disk

Network Interfaces:
eth0 - Internal network (DHCP, e.g. 192.168.10.50)
       Gateway: 192.168.10.1 (router)
eth1 - External network (DHCP, e.g. 10.100.17.65)

Hyper-V:
Internal switch -> eth0
External switch -> eth1
Used as jump host to access internal servers

```bash
sudo apt update
sudo apt upgrade -y

sudo nano /etc/network/interfaces
auto eth0
iface eth0 inet dhcp

auto eth1
iface eth1 inet dhcp
sudo systemctl restart networking

echo "net.ipv4.ip_forward=1" | sudo tee /etc/sysctl.d/ipforwarding.conf
sudo sysctl -p

sudo apt install iptables iptables-persistent -y sudo iptables -t nat -A POSTROUTING -o eth1 -j MASQUERADE sudo iptables-save | sudo tee /etc/iptables/rules.v4 ip a ip route sudo iptables -t nat -L sudo reboot

```

## Configuration as SSH Client

```bash
# Generate SSH key pair (RSA 4096)
ssh-keygen -t rsa -b 4096

# Copy public key to server
ssh-copy-id -i ~/.ssh/id_rsa.pub student@192.168.10.11

# Test SSH login
ssh student@192.168.10.11

# Keys are stored in:
~/.ssh/id_rsa      # Private key
~/.ssh/id_rsa.pub  # Public key
```

## Configuration as LDAP Client 

```bash
# Install
sudo apt update
sudo apt install libnss-ldapd libpam-ldapd ldap-utils -y

# During install:
# URI: ldap://192.168.10.11
# Base: dc=gfn,dc=internal
# Services: passwd, group, shadow

# NSS config
sudo nano /etc/nsswitch.conf
# Ensure:
# passwd: files systemd ldap
# group:  files systemd ldap
# shadow: files ldap

# Auto create home
echo "session optional pam_mkhomedir.so skel=/etc/skel umask=0022" \
| sudo tee -a /etc/pam.d/common-session

# Restart
sudo systemctl restart nslcd
sudo systemctl restart nscd

# Test
getent passwd hwang
id hwang
su - hwang
```
